# 🎬 Moksh Rastogi — Developer Portfolio

A stunning, cinematic software engineer portfolio inspired by the Netflix interface.

![Netflix Portfolio Theme](hero-bg.png)

*A premium digital experience built from the ground up for Moksh Rastogi.*

## ✨ Features

- **Cinematic Hero Section:** Parallax scrolling background with dynamic, typing-effect role display.
- **"Now Streaming" Identity:** Bold red `#E50914` accents, deep solid blacks, and glassmorphism elements to emulate the Netflix UI.
- **Interactive SVG Progres Rings:** Sleek, animated skill rings that draw themselves as you scroll into view.
- **"Top 10" Projects Carousel:** Netflix-style thumbnails featuring match percentages, live demo links, and hover scaling effects.
- **Responsive & Fluid:** Flawless scaling from ultra-wide movie monitors down to mobile screens.
- **Pure Vanilla Magic:** Built completely with HTML5, CSS3, and JavaScript — no heavy frameworks or dependencies!

## 🚀 Live Demo

[**Watch Now 🍿**](https://moksh-rastogi.github.io/netflix-portfolio/)

## 🛠️ Built With

- **HTML5:** Semantic architecture.
- **CSS3:** Custom properties, flexbox/grid layouts, keyframe animations, and filter effects (glassmorphism/blur).
- **Vanilla JavaScript (ES6+):** Intersection Observers for scroll animations, dynamic modal rendering, and toast notifications.

## 📁 Project Structure

```text
📦 PORTFOLIO
 ┣ 📜 index.html    # Main structural layout
 ┣ 📜 style_v3.css  # Custom Netflix-theme stylesheet
 ┣ 📜 script_v3.js  # Interactive features, animations & Intersection Observers
 ┣ 🖼️ avatar.png    # Personalized Netflix-style avatar profile picture
 ┗ 🖼️ hero-bg.png   # Deep cinematic code aesthetic hero background
```

## 💻 Running Locally

1. Clone or download this repository.
2. Open the `index.html` file in your favorite modern browser.
   - *Tip: If you're using VS Code, try the **Live Server** extension for real-time updates while editing.*
3. Enjoy the show!
